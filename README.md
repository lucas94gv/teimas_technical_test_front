# ⚽ Football App – Frontend (React + Vite)

Aplicación frontend desarrollada en React que consume la API de Football App.

---

## 🛠 Tecnologías

- Node.js 18+  
- React 18  
- Vite  
- React Router DOM  
- Axios  

## 🛠 Instalación
```
git clone git@github.com:lucas94gv/teimas_technical_test_front.git
cd my-football-app
npm install
npm run dev
```
